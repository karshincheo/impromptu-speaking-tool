# Impromptu Speaking Tool

A zero-dependency, browser-based practice tool for impromptu speaking (Table Topics, interview prep, debate warm-ups).

**Three pages, no build step — just open in a browser:**

- [`index.html`](index.html) — random topic generator to prompt your speech
- [`timer.html`](timer.html) — speech timer to keep you within time
- [`feedback.html`](feedback.html) — structured feedback form for evaluating speakers

## Use it

```bash
open index.html   # or just double-click it
```

Built as a quick utility for practicing public speaking with friends — plain HTML/CSS/JS, works offline.
